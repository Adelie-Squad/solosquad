## 3. v1.2.5+ 슬롯 (별도 plan)

본 plan 에 *포함 안 됨* — 우선순위 낮거나 별도 작업 단위:

- **`prepublishOnly` 에 seed-path manifest 검증** — v1.2.3 회고의 follow-up. `collectSeeds()` 반환과 `npm pack --dry-run` 출력 교차검증.
- **`apply()` 의 silent seed-skip → abort** — 같은 회고의 follow-up. bundle source 부재 시 *어느 파일이 어디서 누락됐는지* 명시.
- **`scaffoldOrg.copyBundleFile()` 의 silent skip** 도 동일 정합.
- **`doctor --discord` 가 `messenger_user_id` missing 자동 prompt + auto-fix** — 6번째 hop 추가.
- **Slack adapter 복원** — `init` messenger picker 의 Slack 옵션 다시 활성화 + v1.2.x adapter 실 구현.
- **`solosquad chief` CLI group** — `pm` 의 별칭 (v2 SemVer slot 에서 `pm` deprecation 진행 가능).
