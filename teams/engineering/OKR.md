```markdown
### Objective 1
"..."

#### Key Results
- KR1: deploy frequency / lead time / change failure rate / MTTR (DORA 지표)

- KR3: ...
```

## 권장 KR 카테고리

- **DORA 4 metrics** — deploy frequency, lead time, change failure rate, MTTR
- **품질** — test coverage, code-review 통과율, security audit findings
- **속도** — sprint velocity, PR merge time
- **부채** — refactor 작업 비중, deprecated API 사용량
