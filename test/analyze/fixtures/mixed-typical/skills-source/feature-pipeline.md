# Feature Pipeline

Multi-stage workflow for shipping a new feature.

## Stages

Stage 1: Discovery → Stage 2: Planning → Stage 3: Build → Stage 4: Ship

Handoff occurs between each phase. The pipeline expects a structured handoff.md document at each step.

## Phases

- Phase 1: requirements
- Phase 2: design
- Phase 3: implementation
- Phase 4: QA

Each step has a clear input/output contract. The handoff document carries context to the next stage.
